To run the application,
    1) change the directory to Source
    2) open Application.cpp and choose the function to execute
    3) then use the command line,
        i) bash run.sh

        or you can make the script executable by,
        i) chmod +x run.sh 
        ii) ./run.sh
    4) Output will be shown in the terminal.

Source folder consist of all the codes, and the header files

Outputs folder will consist of the goldens(outputs) obtained after running the code, and the expected outputs

Documents folder will consist of the TestPlan and the TestReport for the program created.
